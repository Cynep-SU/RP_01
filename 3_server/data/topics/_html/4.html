Определённый интеграл

Определённый интеграл функции $ y = f(x) $ есть число, значение которого зависит от вида этой функции и пределов интегрирования a и b

\[ \int_a^b f(x) dx \]

a - нижний предел интегрирования b - верхний предел интегрирования

Определённый интеграл вычисляется по формуле Ньютона-Лейбница

\[ \int_a^b f(x) dx = F(x) |_a^b = F(b) + С - F(a) - С = F(b) - F(a) \]

Пример:

\[ \int_1^2 \frac{dx}{x ^ 2} = \frac{x ^ -1}{-1} |_1^2 = 0.5 + 1 = 0.5 \]
